# 🏥 HealthMate - Friendly Medical Assistant

An AI-powered medical assistant chatbot that provides general health information 
and symptom checking using CSV data files.

## 🎯 Features
- 🤒 Symptom checking for 31 common conditions
- ❓ Smart Q&A to confirm symptoms
- 💊 Medicine recommendations
- 🛡️ Precautions and self-care tips
- 🚨 Emergency condition detection
- 📱 Works on mobile and desktop
- 💬 Export conversation history

## 🚀 Quick Start
```bash
pip install -r requirements.txt
streamlit run app.py
